Source for the Twitter bot [@panganBot](https://www.twitter.com/panganBot). It posts daily price of several food item in Indonesia, harvested from [hargapangan.id](http://hargapangan.id/) using [{rvest}](https://rvest.tidyverse.org/), saved to [ElephantSQL](https://elephantsql.com/), diagrams is made using [{ggplot2}](https://ggplot2.tidyverse.org/) and send to twitter using [{rtweet}](https://docs.ropensci.org/rtweet/) and [GitHub Actions](https://docs.github.com/en/actions). Built by [@nurussadad](https://twitter.com/nurussadad). Inspired by [@londonmapbot](https://www.twitter.com/londonmapbot) and [@immunityunity](https://www.twitter.com/immunityunity).
